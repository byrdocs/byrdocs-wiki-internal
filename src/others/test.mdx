## 简答题

### 1.

在前言中，哪些项目是必需的？简要说明这几个项目的格式。

<Solution>
- 时间：xxxx-yyyy学年第[一/二]学期。
- 阶段：期中或期末。
- 类型：本科或研究生。
- 科目：课程的全名。
</Solution>

### 2.

在前言中，“来源”指的是什么？它应该是什么格式的？如何查到这则信息？

<Solution>
“来源”指的是 BYR Docs 主站上的文件，它在题目页中呈现为一个指向主站文件的链接。

它的格式是一个 md5 值（其中字母全用小写），通过这个值可以搜索到主站文件。

要获知一个目标文件的 md5 值，可以在主站查到这个文件，并在文件信息框的右下角复制得到。
</Solution>

### 3.

某份试题有一张题图，与正文文件 `index.mdx` 放在同一个试题目录下，图片文件名为 `题图1.svg`。

如果要使用维基组件定义这张图片（及图片标题），应当如何写？

<Solution>
```mdx
<Figure float src="题图1.svg" background="white">
    BYR Docs 维基真题
</Figure>
```

其中 `float`、`src="题图1.svg"` 和 `background="white"` 的位置可以调换。若图片本身应保持透明，也可以改成 `background="transparent"`。
</Solution>

### 4.

在使用 `Choices` 组件的选择题中，如果所有选项都是 `Option`，那么这题会如何？如果给其中一个 `Option` 加上 `correct` 属性，又会如何？

<Solution>
如果所有选项都是 `Option`，这题会被视为“暂无答案”。

如果给某一选项加上 `correct` 属性，这个选项会变成这题的答案。
</Solution>
